# Memory Inbox

Use this folder for rough notes before they are promoted into source docs, MOCs, glossaries, or templates.

Promotion rule: if a note becomes durable knowledge, move it to the owning source document or a focused memory note. If it stays task-local, move it to ignored `.codex/` task memory instead.
